## This folder contains multiple notebooks we think may be useful for users.

1. `dna_prepare_visualise_data.ipynb` contains all info about used DNA data, visualize data using SMCReader.  

2. `enerf_use_our_colmap.ipynb` contains steps to reproduce our ENERF data using our COLMAP and init point cloud.    

3. `enerf_prepare_colmap_from_scratch.ipynb` **NOT RECOMMENDED!!!** contains all steps to reproduce our full data preprocessing for ENeRF scenes.   

4. `gather_results.ipynb` get mean results across synthetic scenes.   

5. `visualize_normal_conventions.ipynb` compare normal conventions where normals were rendered by different tools (for example Blender vs Gaussian Splatting).
